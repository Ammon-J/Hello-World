# Hello World Erlang

This is a super simple program to demonstrate the basic functions that are offered in Erlang.


## Instructions for Build and Use

Steps to build and/or run the software:

1. Open project in VSCode 
2. Download Erlang from Erlang website (Esay way to do it, is to do it through copilot in VSCode)
3. To compile code run: erlc -o ebin .\src\helloworld.erl in the cmd of VSCode
4. To run the code run: erl -noshell -pa .\ebin -s helloworld hello_world -s init stop

Instructions for using the software:

1. All you have to do is compile and run the code

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:
